Задача 1.
Необходимо создать класс "Дробь" и  добавить интерфейс на 2 метода, также важно кэшировать вещественное значение этой дроби. Для решения задачи создаётся интерфейс FractionMethods объявляющий 2 метода: получение вещественного значения и установка числителя и знаменателя. Внутри класса "Fraction" храняться параметры: числитель, знаменатель, кэшированное вещественное значение и валидация кэша, а также все необходимые конструкторы, методы и т.д.. Для демонстрации решения задачи внутри главного класса Main создаются объекты класса дроби, выводяться их вещественные значения и сравниваются парочка из них.

Задача 2.
Необходимо создать класс "Cat", интерфейс для мяукающих и реализовать подсчёт мяуканий. Для решения задачи создается интерфейс Meowable, объявляющий единственный метод - мяуканье, класс Cat, необходимый по условию класс с методом Funs, а так же декоратор CountMeow, который добавляет новую функциональность для всех "мяукающих" объектов. Для демонстрации решения задачи внутри главного класса Main создаётся объект класса Cat, создаётся декоратор для того же объекта, вызывается несколько раз мяуканье из класса Funs и выводится общее количество мяуканий.

Задача 3.
Необходимо составить программу, которая вставляет в список L за первым вхождением элемента E все элементы списка L, если E входит в L. Для решения задачи создаёт класс, содержащий только метод для вставки копии списка после определённого значения Е. Для демонстрации решения задачи внутри главного класса Main сначала создается список, содержащий целые числа, и преобразуется через выбор определённого значения, затем тоже самое происходит с созданным списком, содержащим буквы Латинского алфавита.
