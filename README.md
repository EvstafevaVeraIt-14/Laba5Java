Задача 1.
Необходимо создать класс "Дробь" и  добавить интерфейс на 2 метода, также важно кэшировать вещественное значение этой дроби. Для решения задачи создаётся интерфейс FractionMethods объявляющий 2 метода: получение вещественного значения и установка числителя и знаменателя. Внутри класса "Fraction" храняться параметры: числитель, знаменатель, кэшированное вещественное значение и валидация кэша, а также все необходимые конструкторы, методы и т.д.. Для демонстрации решения задачи внутри главного класса Main создаются объекты класса дроби, выводяться их вещественные значения и сравниваются парочка из них.

Для демонстрации решения задачи внутри главного класса Main создаётся объект класса Cat, создаётся декоратор для того же объекта, вызывается несколько раз мяуканье из класса Funs и выводится общее количество мяуканий.

Для решения задачи сначала создаём классы: Point, Line и PolyLine.
